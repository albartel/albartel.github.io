dt_train_acc = []

Part 9: Create and Evaluate Final Model¶

The final model that I will select is a random forest model, given that it had the highest validation accuracy of the three at 0.8677, compared to a maximum of 0.772 for the optimal decision tree model, and 0.6111 for the logistic regression model. Of the different options for hyperparameters for the random forest model, I determined that the value for max_depth that had the highest validation accuracy was 28, so that is the value that will be used in the final model. For the n_estimators hyperparameter I will continue to use 100.

In [22]:
np.random.seed(1)
